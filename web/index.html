ON / OFF

ON で最適化、OFF で Windows 既定へ。ON to optimize, OFF to restore Windows defaults.

切り替えられる設定はすべてスイッチ。ON = 最適化を適用 / OFF = Windows 既定に戻すで統一されているので、試して合わなければいつでも元に戻せます。レジストリや COM を C# から直接制御するので、コマンド文字列をそのまま流し込むより安全・確実です。Every toggleable setting is a switch. It's consistent — ON = apply the optimization / OFF = back to Windows default — so if a tweak doesn't suit you, flip it back anytime. Settings are driven natively from C# (registry / COM), which is safer and more reliable than piping raw command strings.
