# Lumin4ti

**散らかった Windows を、ひと灯りで整える。**

Lumin4ti は、Windows 10 / 11 のメンテナンスや設定変更を、分かりやすい画面から行える無料アプリです。コマンドを入力する必要はなく、各項目の効果と注意点を確認しながら操作できます。

[Lumin4ti をダウンロード](https://lumin4ti.nephilim.jp/Lumin4ti-win.msi)（Windows 64-bit）

## できること

- インストール済みアプリや Windows セキュリティの更新・修復
- Windows Update の不要ファイル、壊れたスタートアップ項目、キャッシュなどの整理
- メモリ、バックグラウンド動作、電源、描画などのパフォーマンス調整
- マウス、メニュー、フォルダー表示、アクセシビリティなどのシステム設定
- クイックアクセスのピン留め (パス昇順) や環境変数 Path の並べ替え

このほかにも、Windows のメンテナンスに役立つ機能をカテゴリ別にまとめています。

## インストール

1. [署名済みインストーラー](https://lumin4ti.nephilim.jp/Lumin4ti-win.msi)をダウンロードします。
2. ダウンロードした `Lumin4ti-win.msi` を実行し、画面の案内に従います。
3. Lumin4ti を起動し、ユーザーアカウント制御（UAC）が表示されたら内容を確認して許可します。

新しいバージョンがある場合は、アプリが更新を案内します。

## 使い方

1. 目的に合うカテゴリを開きます。
2. 項目に表示される効果と注意点を確認します。
3. ボタンを押すか、スイッチを切り替えて適用します。

多くのスイッチ項目は、**ON で設定を適用し、OFF で適用前の状態または Windows の既定へ戻せます**。項目によって動作が異なるため、画面の説明を確認してから操作してください。

## 利用前の注意

- システム設定を変更するため、管理者権限が必要です。
- 実行ボタンの操作には、削除やリセットなど元に戻せないものがあります。必要なデータは事前にバックアップしてください。
- 再起動が必要な項目は、画面にその旨が表示されます。
- 処理中はアプリを終了したり、PC の電源を切ったりしないでください。

## 動作環境

- Windows 10 / 11（64-bit）
- 管理者権限
- ダウンロード、更新、一部機能の利用にインターネット接続が必要

追加のランタイムをインストールする必要はありません。日本語・英語を含む 17 言語に対応しています。
