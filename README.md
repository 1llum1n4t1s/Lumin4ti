# Lumin4ti

Windows のメンテナンス・最適化をワンクリックで行えるツールです。コマンドプロンプトや PowerShell を開かなくても、よく使うシステム調整を画面から安全に実行できます。ON/OFF を切り替えられる設定はスイッチになっていて、いつでも Windows 既定の状態に戻せます。

## 主な機能

### 更新・セキュリティ
- **アプリの一括更新 (winget)** — パッケージ ID とアプリ名を厳密に照合し、インストール済みアプリを安全にまとめて更新
- **Defender 定義ファイルの再取得** — 定義破損時の定番リフレッシュ
- **Defender 設定のリセット** — 除外パスなどの「スキャンの抜け穴」を一掃
- **Windows セキュリティ アプリのリセット** — 設定画面の不調を修復

### クリーンアップ・修復
- **Windows Update の使用スペースを削除** — WinSxS の古い更新を削除して数 GB 回収
- **システムフォルダ名の破損を修復** / **トレイアイコンのキャッシュをリセット**
- **リンク切れスタートアップ・孤立した関連付けの掃除**
- **イベントログの全削除** / **Store キャッシュのクリア** / **SSD TRIM と空き領域の統合**

### パフォーマンス(多くは ON/OFF スイッチ)
- **MMAgent の最適化**(メモリ圧縮・ページ結合・プリフェッチ)
- **VBS / HVCI 無効化**(メモリ節約 1〜2GB。Hyper-V / WSL2 利用時は OFF 推奨)
- **SvcHost プロセスの統合** / **UWP バックグラウンド一括オフ・元の個別設定へ復元** / **Edge 常駐の無効化**
- **Windows Recall の無効化** / **電力スロットリング無効化** / **MPO 無効化**
- **オーディオ優先度の引き上げ** / **RDP 60FPS 化** / **スタートアップ起動遅延の解除**
- **NVIDIA GPU の割り込み分散の最適化**

### システム設定(多くは ON/OFF スイッチ)
- **電源プラン / 電源モードの推奨構成** / **休止状態の無効化**
- **マウス加速の無効化** / **固定キーなどのホットキー無効化**
- **メニュー表示遅延の短縮** / **フォルダテンプレートの汎用固定**
- **CEIP / フィードバック通知の無効化** / **Game Bar 設定の復元**
- **NVMe ドライバ上書きの解除** / **時刻同期サーバを国内 NTP に設定**

### 整理・ソート
- **クイックアクセスのピン留めを名前順にソート**(実行前に自動バックアップ)
- **環境変数 Path のソート**(重複除去 + 昇順)

## 動作環境

- Windows 10 / 11
- **管理者権限が必要です**(起動時に UAC の確認が表示されます)

## 多言語対応

17 言語に対応しています(日本語 / English / Deutsch / Español / Français / Italiano / Português (Brasil) / Bahasa Indonesia / Filipino / Русский / Українська / 简体中文 / 繁體中文 / 한국어 / தமிழ் / संस्कृतम् / Latina)。初回起動時は Windows の表示言語から自動選択され、「バージョン情報」タブの「言語」から切り替えられます(選択は保存され、次回以降も維持されます)。

## 使い方のヒント

- 各項目には効果と注意点の説明が書かれています。迷ったら説明を読んでから切り替えてください
- スイッチの **ON = 最適化を適用 / OFF = Windows 既定に戻す** で統一されています
- 「※ 反映には PC の再起動が必要です」の項目は再起動後に有効になります
- エクスプローラーに影響する変更は、実行後に自動でエクスプローラーを再起動します

## インストール

署名済みPerMachine MSI ([`Lumin4ti-win.msi`](https://lumin4ti.nephilim.jp/Lumin4ti-win.msi)) をダウンロードして実行してください。Lumin4tiは保護された`Program Files`へインストールされ、以後は自動更新機能が最新版を通知します（「バージョン情報」タブから手動確認も可能）。

旧`%LocalAppData%\Lumin4ti`版を使用している場合は、次回起動時に署名済みMSIへの移行確認が表示されます。設定・ログ・復元用バックアップを保持したまま移行し、完了後に旧本体・Updater・パッケージキャッシュ・旧ショートカットを回収します。

## 設定・ログの保存先

- 設定: `%APPDATA%\Lumin4ti\settings.json`
- ログ: `%APPDATA%\Lumin4ti\logs\`
- 管理者設定の復元用バックアップ: `%ProgramData%\Lumin4ti\backups\`（Administrators / SYSTEM のみ変更可能）

クイックアクセスの並び替えバックアップは、対象 Jump List と同じユーザープロファイル内に `.lumin4tibak` として保存されます。

「バージョン情報」タブのボタンからも各フォルダを開けます。
